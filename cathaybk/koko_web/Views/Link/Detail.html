國泰世華商業銀行數位存款帳戶特別約定條款

存戶茲以網路方式向國泰世華商業銀行(以下簡稱　貴行)申請開立新臺幣活期存款帳戶(以下簡稱「數位存款帳戶」、「本帳戶」)，除已知悉並明瞭貴行於「國泰世華商業銀行【存款暨理財】綜合約定書」(以下簡稱「綜合約定書」，含嗣後經依約修改或變更之條款)中之重要告知事項及個人資料保護法之法定告知事項外，並依數位存款帳戶之功能與服務，遵守前開綜合約定書中之各該相應條款及「數位存款帳戶特別約定條款」(以下簡稱「本約款」)之各該約款，本約款與前開綜合約定書之約款不符者，優先適用本約款之約定：

  • 用詞定義

    • 數位存款帳戶：指存戶以網路方式 ( 包含但不限於網路銀行、行動銀行或自動化設備 ( ATM ) ) 向貴行申請開立之新臺幣活期存款帳戶。

    • 存戶：指接受貴行提供數位存款帳戶服務且具有本國國籍之成年自然人。

    • 概括約定：係指存戶與　貴行事先約定本人某一轉出帳戶可透過網路銀行、行動銀行或自動化設備 ( ATM ) 發動交易指示，繳納　貴行所提供之公用事業費及各類稅費服務。

  • 本帳戶開戶程序

    • 數位存款帳戶採實名制，存戶應提供身分基本資料 ( 包含但不限於姓名、國民身分證統一編號、國民身分證領補換資料、出生年月日、聯絡方式、開戶目的與性質)、國民身分證、第二身分證明文件(如健保卡等)正面影像檔；惟若開立前條1-1類數位存款帳戶者，尚須留存人像影像。若存戶於開戶申請完成前中斷，存戶同意申請資料將暫存二日，存戶可經由一定驗證後接續前次申請流程完成開戶。若未於二日內完成開戶申請，貴行將清除暫存資料，暫存資料將不作其他使用。

    • 存戶申請資料如有誤寫或其他顯然錯誤者，存戶同意　貴行得逕行或向存戶確認後更正。存戶聯絡方式倘有變更者，應即通知貴行。貴行對存戶所為之通知或函件，依存戶國民身分證所載地址或最後提供地址、電子郵件信箱為寄發後，經通常傳遞期間即視為已送達生效。若因存戶提供聯絡方式錯誤或未主動通知貴行聯絡方式變更者，所衍生利息、違約金或其他損失，由存戶自行負責，概與　貴行無涉。

    • 存戶申請資料如有誤寫或其他顯然錯誤者，存戶同意貴行得逕行或向存戶確認後更正。存戶聯絡方式倘有變更者，應即通知貴行。貴行對存戶所為之通知或函件，依存戶國民身分證所載地址或最後提供地址、電子郵件信箱為寄發後，經通常傳遞期間即視為已送達生效。若因存戶提供聯絡方式錯誤或未主動通知貴行聯絡方式變更者，所衍生利息、違約金或其他損失，由存戶自行負責，概與貴行無涉。

    • 若審查程序未通過(如身分證影像模糊、證件錯誤等)，申請資料將由貴行依法留存，並由貴行通知存戶進行必要之程序。存戶同意配合貴行於收到通知後三日內補正資料或補齊文件，若逾期未完成，存戶瞭解須本人攜帶身分證及第二身分證明文件至貴行任一分行辦理帳戶啟用或銷戶程序。

    • 存戶向 貴行申請核發之晶片金融卡，經貴行審查程序通過後，得以郵寄方式寄送至存戶指定之地址或由存戶親至臨櫃申請，存戶於收到金融卡後須至貴行 ATM 完成開卡，始得啟用卡片功能。貴行提供其他得下載金融卡功能之服務者 ( 例手機金融卡或同卡多帳戶服務功能等 )，存戶得經 貴行同意後，選擇以該等方式使用晶片金融卡之各項服務。

  • 本帳戶銷戶程序

    • 以臨櫃方式受理結清銷戶
      應由存戶本人親自辦理，本人如因特殊情況無法親自辦理結清銷戶時得委託代理人為之，該代理人除應出具授權書外，銀行並應確
      認本人及代理人之身分。

    • 以郵寄方式受理結清銷戶存戶以郵寄結清銷戶申請書方式辦理時，帳戶餘額以不超過新台幣壹拾萬元（或等值外幣）為限，經 貴行以電話或其他方式確認存戶身分，並完成下列審核手續後，辦理結清銷戶：
      （一）帳戶仍有餘額者， 貴行得於扣除相關費用後，依申請書指示之方式辦理，存戶得選擇開立本人抬頭且劃線禁止背書轉讓之支
      票或指示匯入本人之其他帳戶。存戶如選擇將帳戶餘額匯入其於他行之帳戶， 貴行得請存戶提供該帳戶之存摺封面影本或其他證明文件以供確認。
      （二）結清銷戶基準日係以銀行完成銷戶手續當日為準。

    • 以網路方式受理結清銷戶
      存戶以網路結清銷戶方式申辦時，帳戶餘額以不超過新台幣伍萬元（或等值外幣）為限。
      貴行應以顯著之方式，於網站網頁上揭露結清銷戶相關內容，具供存戶審閱、點選「同意」及確認功能，系統並應具備檢核機制。貴行並得以適當方式確認存戶身分。存戶結清活期帳戶仍有餘額者，銀行得於扣除相關費用後，依存戶指示匯入本人之其他帳戶，結清銷戶基準日係以 貴行電腦系統完成銷戶當日為準。

  • 本帳戶臨櫃服務及相關服務之申請

    • 存戶擬就數位存款帳戶申請使用 貴行提供之臨櫃服務者，於首次臨櫃辦理除存款以外之業務時，應攜帶本人國民身分證及第二身分\證明文件，並填寫印鑑卡留存約定往來印鑑樣式(簡稱「原留印鑑」、「原留印鑑樣式」），「原留印鑑樣式」為嗣後存戶臨櫃辦理業務之查核依據。存戶設定臨櫃取款密碼者，則得使用 貴行提供之臨櫃無摺取款服務。 原留印鑑、密碼及其他取款憑證等存戶應妥善保管及保密，凡以存戶之原留印鑑樣式取（匯）款或向　貴行申辦、變更或中止與相關服務項目者，均視為存戶之代理人，故存戶之原留印鑑、密碼及其他取款憑證如有遺失、滅失、被竊、被搶或其他情事而脫離占有時，應於營業時間內親自或以電話向　貴行辦理掛失止付相關手續，但原留印鑑、密碼及其他取款憑證之掛失限向原開戶單位辦理，但經　貴行同意受理者，不在此限。在　貴行受理完成掛失止付手續以前，已經付款或受理申辦者，對存戶仍有清償或申辦事項生效之效力。但以電話掛失者，應於次一營業日前補辦書面手續。

    • 存戶於辦理前條第一項程序時由貴行以拍照方式建立影像檔完成留存紀錄者，原數位存款帳戶之類型即變更為第一類(1-1類)，實際提供服務依 貴行相關規定辦理。

    • 存戶申請網路銀行、行動銀行及理財密碼者，已瞭解須登入網路銀行、行動銀行完成帳號及密碼設定後始得使用。

    • 存戶向貴行申請將數位存款帳戶轉換為一般存款帳戶時，應依 貴行開戶流程辦理，並經 貴行製發存摺交付後完成轉換。

    • 存戶未申辦數位存款帳戶之臨櫃無摺取款服務者，應以 貴行製發之晶片金融卡於自動化設備存提領款項。

    • 存戶完成前條第一項、第二項及(或)第三項之程序者，得依與 貴行申請使用之各項服務功能內容辦理。

    • 存戶瞭解得與貴行概括約定數位存款帳戶繳納　貴行所提供之公用事業費及各類稅費服務。詳細約定悉依各項服務授權條款辦理。

    • 存戶已向貴行申請將數位存款帳戶轉換為一般存款帳戶者，另依一般存款帳戶之約定方式辦理。

    數位存款帳戶適用新臺幣活期儲蓄存款牌告利率。有關款項存提領、利息計付、查詢、密碼變更、帳戶暫停、恢復、終止使用及結清銷戶之方式，悉依國泰世華銀行[存款暨理財]綜合約定書第二章第壹節[共同約定事項]辦理。

  • 帳戶使用原則

    • 數位存款帳戶僅限存戶本人使用。

    • 數位存款帳戶如供非法使用存戶應負法律責任。

    • 同一存戶最多開立二戶數位存款帳戶(包含已啟用、審核中、未啟用、及轉換成一般帳戶等狀態之帳戶)。

    • 帳戶查詢、密碼變更及結清銷戶，存戶同意配合貴行核對存戶身分及辦理相關程序。

    • 存戶可依相關規定執行約定轉入帳戶轉帳及非約定轉入帳戶轉帳。

    • 有關數位存款帳戶遭盜用之處理方式、交易憑證(包含但不限於網路銀行、行動銀行理財密碼、隨身密碼、臨櫃提款密碼、金融卡或金融卡密碼)使用方式及遺失、滅失、被竊或遭偽冒、變造時之處理方式及權利義務關係，悉依國泰世華銀行[存款暨理財]綜合約定書第三章[附加功能約定事項]辦理。
