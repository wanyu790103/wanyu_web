KOKO 問答集

解答疑惑 KOKO 有責！

    • 申請 KOKO 的相關服務，請點選「立即申辦」，下載 KOKO APP 後進行註冊即可。

    • KOKO 帳戶申請資格如下：

      1. 限年滿 20 歲的本國自然人，提供身分證、第二證件(健保卡、駕照、護照...)！
        若為以下二種身分，還需要準備自然人憑證或至國泰分行進行臨櫃驗身才可完成申請流程。

        1. 本行新戶

        2. 本行卡友

      2. 暫不接受未滿 20 歲的未成年人申請數位帳戶。

      3. 暫不接受外國人申請數位帳戶。

      4. 每人最多開立三戶數位存款帳戶，惟 KOKO 通路限辦一戶

      以上文件都備齊後，就可以到 KOKO APP進行線上開戶流程囉！

    • 依據開戶身分認證方式不同， KOKO 帳戶分為兩種類型：
      第一類：新客戶 、本行信用卡卡友，可透過自然人憑證線上完成認證，或至分行臨櫃驗證身分。
      第二類：本行存戶，使用既有帳號連結的簡訊驗證碼驗證身分。
      ※如需進行公共事業代扣繳、跨國提款、臨櫃取款等業務，可至分行驗證後升級 1-1 類(超級)帳戶。

    • 可能的原因有以下兩種：

      1. 你是本行卡友或存戶，但此次輸入的資料與本行留存資料不符。

      2. 你是本行存戶且持有本行信用卡，但信用卡資料與開戶資料不相同。

    • 一天最多可發送 10次簡訊（含驗證通過及失敗次數）。
      若當日已達 10 次，就無法進行線上開戶了，請等到隔日（凌晨 12 時過後）再重新進行。

      1. 請將 https://www. KOKO bank.com 加入「信任的網站」( Internet Explorer〉工具〉 網際網路選項〉 安全性〉 信任的網站〉 網站 )

      2. 回到「網際網路」視窗後，將此區域的安全性等級設定為低。

      3. JAVA 元件需更新，請至 JAVA 官網進行下載與安裝。

      ※ 線上開戶最適操作環境為 Microsoft Windows XP 以上作業系統，瀏覽器 Safari、IE 11、Firefox 3 以上，請更新瀏覽器版本

    • 請耐心等候一下唷！
      完成申請流程後，需 2-3 個工作天進行帳戶審核，若帳戶或申請證件有任何問題，將以 Email 及簡訊通知客戶。

    • KOKO 帳戶狀態若顯示為「帳戶審核中」，代表你已經成功進件，約需兩個工作天的審核時間，請留意帳戶審核結果將寄發至你的 email 信箱。

      1. 證件拍攝模糊不清、有光點

      2. 身分證拍攝同一面(需拍攝正反面)

      3. 證件為影本

    • 當帳戶狀態為「待補件」時，請在 7 天之內（含假日）重新上傳證件影像，將在1-2個工作天重新審核帳戶。
      若超過線上補件期限，待帳戶狀態變成「未正式啟用」時，請持雙證件親至國泰世華分行辦理補件，帳戶即可正常啟用。

    • KOKO 一卡通 i 金融卡可線上申辦，具備提款轉帳和刷卡消費功能。

    • 若你是線上申請 KOKO 金融卡，收到卡片後，請依以下方式進行開卡：

      1. KOKO APP：點選我的帳戶 → 選擇帳戶管理 → 點選寄送密碼。

      2. 國泰世華網銀 MyBank → 銀行服務 → 存款帳戶查詢 → 功能選擇 → 帳戶狀態 → 領取開卡密碼。

      至電子郵件信箱收取密碼，再至 ATM 進行開卡步驟。

    • 利用轉帳轉入、ATM 存入現金或臨櫃匯款都可以喔!

      1. 致電客服：驗證理財密碼後方可掛失補發。

      2. 臨櫃申請：若您未申請或忘記理財密碼，請臨櫃進行掛失補發。

      掛失補發每卡將收取手續費新臺幣 100 元。

    • 不行唷！KOKO COMBO icash聯名卡只能透過 KOKO APP 線上申請，無法臨櫃申辦的。

    • 不分卡別，首年免年費，次年起申辦電子帳單也免年費唷！

    • 透過 KOKO APP 轉帳(手機通訊錄/通訊軟體)，在取款產生後 24 小時內需要取款完成，不然這筆交易將自動失效。

    • 若是透過 KOKO 手機通訊錄或通訊軟體轉帳，在取款產生後 24 小時內且取款人未取款前，可以在 KOKO APP 透過訊息 → 轉帳訊息 → 點選交易取消，此筆太好轉交易就取消囉！

      若是直接使用銀行帳戶或 KOKO 好友，則是立即轉出，無法取消喲！

    • KOKO 轉帳的取款密碼並無法做變更或查詢喔！若忘記取款密碼，請取消此筆交易，再建立新的就可以了。

    • 如果 KOKO 拒絕付款，發送通知的人只會接到拒絕轉帳通知，並不會有其他影響。

    • 您可以使用其他銀行帳號轉帳到 KOKO 帳號，流程與一般轉帳相同，惟對方會收取跨行轉帳手續費 15 元。

    • 請你先至帳戶明細確認是否有該筆轉帳記錄，若無該筆轉帳記錄，請重新再操作一次轉帳流程。
